# 植物奶与素奶酪的制作原理

> "The best plant milk is the one you make yourself." —— 最好的植物奶是你自己制作的。

自制植物奶和素奶酪不仅成本更低、更健康，还能让你完全控制食材和口味。了解制作原理后，你就能创造出属于自己的独特配方。

## 植物奶的制作原理

### 基本原理

植物奶的制作本质上是一个**提取、乳化、过滤**的过程：

1. **浸泡**：软化植物原料，释放营养素
2. **研磨/搅拌**：将植物材料破碎，释放蛋白质和脂肪
3. **过滤**：分离固体残渣和液体
4. **调味**（可选）：添加甜味剂、香草、盐等

### 通用制作流程

**基础配方：**
- 1杯坚果/种子/谷物（干重）
- 3-4杯水
- 可选：2-3个椰枣（去核）或1-2汤匙枫糖浆
- 可选：1茶匙香草精
- 可选：一小撮盐（提升风味）

**步骤：**

1. **浸泡**（8-12小时）
   - 将坚果/种子浸泡在水中
   - 软化纤维，提高营养吸收率
   - 某些食材（如燕麦）只需30分钟

2. **冲洗**
   - 倒掉浸泡水
   - 用清水冲洗干净

3. **搅拌**
   - 将食材与新鲜水放入高速搅拌器
   - 高速搅拌1-2分钟，直至顺滑

4. **过滤**
   - 使用坚果奶袋或细纱布过滤
   - 挤压出所有液体

5. **储存**
   - 冷藏保存，3-5天内饮用
   - 使用前摇匀（会自然分层）

## 不同植物奶的制作

| 植物奶 | 特点 | 制作要点 | 营养价值 | 常见问题/技巧 |
|:------:|:----|:--------|:--------|:------------|
| **杏仁奶** | 口感清淡，蛋白质丰富，适合通用 | 使用生杏仁（无盐、无烤制）；浸泡时间：8-12小时；水与杏仁比例：3:1 或 4:1；可以添加椰枣增加甜味 | 富含维生素E、钙、健康脂肪；低热量（自制版本） | - |
| **燕麦奶** | 质地浓厚，天然甜味，成本低 | 使用传统燕麦（非即食燕麦）；浸泡时间：仅需30分钟；不要过度搅拌（会产生粘性）；水与燕麦比例：3:1 | 富含β-葡聚糖（降低胆固醇）；高纤维、低脂肪 | **太粘稠**：减少搅拌时间，或使用较少燕麦；**分层太快**：添加少量葵花籽油或菜籽油 |
| **豆奶** | 蛋白质含量最高，接近牛奶营养 | 使用干大豆（非罐头）；浸泡时间：8-12小时；**需要煮沸**：去除抗营养物质；可以添加香草掩盖"豆腥味" | 完整蛋白质（含所有必需氨基酸）；富含异黄酮（植物雌激素） | 传统方法：先磨成豆浆，再过滤、煮沸；现代方法：搅拌后煮沸，再过滤 |
| **腰果奶** | 最丝滑，无需过滤（可选） | 浸泡时间：2-4小时即可；可以不过滤（如果搅拌机足够强劲）；质地最接近全脂牛奶 | 富含健康脂肪和蛋白质；适合制作奶油类产品 | - |
| **椰奶** | 高脂肪，椰香味，适合烹饪 | 使用无糖椰丝；浸泡时间：30分钟；可以制作浓稠或稀薄版本 | 中链脂肪酸（MCT），易于代谢；适合制作甜品和咖喱 | - |
| **米奶** | 最清淡，过敏友好 | - | - | - |
| **大麻籽奶** | 富含Omega-3 | - | - | - |
| **葵花籽奶** | 成本低，营养丰富 | - | - | - |
| **核桃奶** | 富含Omega-3，口感浓郁 | - | - | - |

## 素奶酪的制作原理

### 基本原理

素奶酪的制作主要依靠：

1. **坚果/种子作为基础**：提供脂肪和蛋白质
2. **发酵**（可选）：产生"奶酪"风味
3. **凝固剂**：如琼脂、卡拉胶、柠檬汁
4. **调味**：营养酵母、盐、香草等

### 基础素奶酪配方

**软质素奶酪（如奶油奶酪）：**

```
材料：
- 1.5杯浸泡的腰果（或杏仁）
- 1/4杯水
- 2汤匙柠檬汁
- 1茶匙盐
- 1汤匙营养酵母（可选）
- 1瓣大蒜（可选）

步骤：
1. 浸泡腰果4-6小时
2. 冲洗干净
3. 所有材料放入搅拌机
4. 高速搅拌至完全顺滑
5. 冷藏2小时以上
```

**硬质素奶酪（如切达）：**

```
材料：
- 1杯浸泡的腰果
- 1/4杯水
- 1汤匙营养酵母
- 2茶匙琼脂粉
- 1茶匙盐
- 1/2茶匙大蒜粉
- 1/2茶匙洋葱粉

步骤：
1. 除琼脂外，所有材料搅拌成泥
2. 琼脂粉与1杯水混合，小火加热至溶解
3. 将坚果泥倒入琼脂溶液中
4. 继续加热搅拌2-3分钟
5. 倒入模具，冷藏4小时以上
```

### 发酵素奶酪

使用**益生菌胶囊**或**开菲尔菌种**可以制作发酵素奶酪：

**步骤：**
1. 制作基础坚果奶或腰果奶油
2. 添加益生菌（1-2粒胶囊内容物）
3. 在温暖处发酵12-24小时
4. 冷藏后使用

**特点：**
- 产生类似奶酪的酸味
- 富含益生菌
- 适合制作素酸奶、素奶油奶酪

## 常见问题与解决方案

### 植物奶分层

**原因：** 天然现象，脂肪和水分分离

**解决：**
- 使用前摇匀
- 添加少量乳化剂（如葵花籽卵磷脂）
- 添加少量油（如葵花籽油）

### 植物奶太淡

**解决：**
- 增加原料比例（如2:1而不是3:1）
- 减少水用量
- 过滤时不要过度挤压

### 素奶酪太硬

**解决：**
- 减少琼脂用量
- 增加水或油
- 使用软质奶酪配方

### 素奶酪没有"奶酪味"

**解决：**
- 增加营养酵母用量
- 添加发酵步骤
- 添加少量白醋或柠檬汁

## 进阶技巧

### 1. 制作素黄油

使用**椰子油 + 坚果**可以制作素黄油：

```
材料：
- 1/2杯椰子油（融化）
- 1/4杯腰果或杏仁
- 1茶匙营养酵母
- 1茶匙盐
- 1/2茶匙姜黄（可选，上色）

步骤：
1. 所有材料搅拌至顺滑
2. 倒入模具
3. 冷藏至凝固
```

### 2. 制作素酸奶

使用**植物奶 + 益生菌**：

```
材料：
- 4杯植物奶（如豆奶、椰奶）
- 2粒益生菌胶囊
- 2汤匙枫糖浆（可选）

步骤：
1. 加热植物奶至体温（37°C）
2. 添加益生菌和甜味剂
3. 在温暖处发酵8-12小时
4. 冷藏后使用
```

### 3. 制作素芝士蛋糕

使用**腰果 + 椰奶**制作免烤芝士蛋糕：

- 底：坚果 + 椰枣 + 可可粉
- 馅：腰果 + 椰奶 + 柠檬汁 + 甜味剂
- 冷藏4小时以上

## 储存与保质期

### 植物奶

- **冷藏**：3-5天
- **冷冻**：可冷冻冰块，用于冰沙
- **注意**：自制无防腐剂，需尽快饮用

### 素奶酪

- **软质**：冷藏5-7天
- **硬质**：冷藏1-2周
- **发酵型**：冷藏1周，味道会随时间变化
