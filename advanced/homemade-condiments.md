# 自制调味品与发酵食品

> "Fermentation is the oldest form of food preservation and flavor enhancement." —— 发酵是最古老的食物保存和风味提升方式。

自制调味品和发酵食品不仅能让你完全控制食材，还能带来独特的风味和健康益处。从基础的酱油、醋到复杂的味噌、泡菜，掌握这些技能将大大提升你的素食烹饪水平。

## 发酵的基础原理

### 什么是发酵？

发酵是**微生物（如细菌、酵母、霉菌）将食物中的糖和淀粉转化为酸、酒精或气体**的过程。这个过程不仅改变食物的风味和质地，还能：

- **增加营养价值**：产生益生菌、B族维生素
- **提高消化率**：分解抗营养物质
- **延长保质期**：酸性环境抑制有害细菌
- **增强风味**：产生复杂的鲜味（Umami）

### 发酵的类型

1. **乳酸发酵**：产生乳酸（如泡菜、酸菜）
2. **酒精发酵**：产生酒精（如米酒、醋的前身）
3. **醋酸发酵**：产生醋酸（如各种醋）
4. **霉菌发酵**：使用霉菌（如味噌、酱油）

## 基础调味品的制作

### 1. 素食酱油（Soy Sauce Alternative）

**原理：** 使用豆类、谷物发酵产生类似酱油的鲜味

**基础配方：**

```
材料：
- 1杯干大豆（或黑豆）
- 1杯小麦（或大麦）
- 盐
- 水

步骤（简化版）：
1. 大豆和谷物分别煮熟
2. 混合后冷却至体温
3. 接种曲霉菌（或使用市售曲）
4. 在温暖潮湿处发酵2-3天（产生"曲"）
5. 加盐水继续发酵3-6个月
6. 压榨、过滤、巴氏杀菌
```

**注意：** 传统酱油制作复杂，建议初学者从购买高质量素食酱油开始，或使用简化版配方。

**快速替代方案：**
- 使用**椰子氨基酸**（Coconut Aminos）作为酱油替代
- 或制作**简易酱油**：使用蘑菇、海带、豆类煮制

### 2. 素食醋（Vinegar）

**原理：** 酒精发酵后，醋酸菌将其转化为醋酸

**水果醋制作：**

```
材料：
- 水果（苹果、梨、葡萄等）
- 糖
- 水
- 醋酸菌母（可选，可从现成醋中获取）

步骤：
1. 水果切块，加糖和水
2. 覆盖纱布，在温暖处发酵1-2周（产生酒精）
3. 添加醋酸菌母
4. 继续发酵2-4周（产生醋酸）
5. 过滤、装瓶
```

**米醋制作：**
- 使用糯米或大米
- 先制作米酒，再转化为醋
- 需要3-6个月

### 3. 味噌（Miso）

**原理：** 大豆和谷物通过曲霉菌发酵产生鲜味

**基础味噌配方：**

```
材料：
- 2杯干大豆
- 1杯米曲（或大麦曲）
- 1/2杯盐
- 水

步骤：
1. 大豆浸泡过夜，煮熟至软烂
2. 冷却后与曲和盐混合
3. 捣碎成泥状
4. 装入密封容器，压实，表面撒盐
5. 在阴凉处发酵6个月至2年
6. 时间越长，颜色越深，味道越浓
```

**类型：**
- **白味噌**：发酵时间短（6个月），甜味
- **红味噌**：发酵时间长（1-2年），咸味重
- **大麦味噌**：使用大麦曲，风味浓郁

**使用建议：**
- 做汤底：1汤匙味噌 + 1杯水
- 调味：添加到炖菜、酱料中
- 注意：不要煮沸，会破坏风味和益生菌

### 4. 素食酸奶（Plant-based Yogurt）

**原理：** 益生菌在植物奶中发酵产生酸味和质地

**基础配方：**

```
材料：
- 4杯植物奶（豆奶、椰奶、燕麦奶）
- 2粒益生菌胶囊（或1/4杯现成酸奶）
- 可选：2汤匙枫糖浆或椰糖

步骤：
1. 加热植物奶至37-40°C（体温）
2. 添加益生菌和甜味剂
3. 倒入干净容器
4. 在温暖处（25-30°C）发酵8-12小时
5. 冷藏后使用
```

**技巧：**
- 使用**酸奶机**或**保温箱**保持温度
- 发酵时间越长，酸味越重
- 可以添加果泥、香草增加风味

### 5. 泡菜（Kimchi / Sauerkraut）

**原理：** 乳酸菌将蔬菜中的糖转化为乳酸

**基础泡菜配方：**

```
材料：
- 1个大白菜（或卷心菜）
- 2汤匙盐
- 可选：胡萝卜、萝卜、辣椒粉

步骤：
1. 白菜切块，用盐腌制2-3小时
2. 冲洗多余盐分
3. 与其他蔬菜混合
4. 装入发酵罐，压实（排除空气）
5. 在室温下发酵3-7天
6. 转移到冰箱继续发酵（可选）
```

**关键点：**
- **压实**：确保蔬菜被盐水覆盖，排除空气
- **温度**：室温20-25°C最适合
- **时间**：根据个人口味调整（越久越酸）

### 6. 营养酵母（Nutritional Yeast）

虽然营养酵母通常是购买的，但了解其用途很重要：

**特点：**
- 富含B族维生素（特别是B12，如果强化）
- 具有"奶酪"和"坚果"风味
- 适合撒在食物上或加入酱料

**使用建议：**
- 制作"素奶酪"酱：营养酵母 + 坚果 + 柠檬汁
- 撒在意大利面、爆米花上
- 添加到汤和炖菜中增加鲜味

## 进阶发酵食品

### 1. 康普茶（Kombucha）

**原理：** 红茶菌（SCOBY）发酵甜茶产生酸味和气泡

**基础配方：**

```
材料：
- 1升红茶或绿茶
- 1/2杯糖
- SCOBY（红茶菌母）
- 1/4杯现成康普茶（作为启动液）

步骤：
1. 煮茶，加糖溶解
2. 冷却至室温
3. 倒入发酵罐，添加SCOBY和启动液
4. 覆盖纱布，在温暖处发酵7-14天
5. 装瓶，二次发酵2-3天（产生气泡）
6. 冷藏后饮用
```

**注意：** 需要保持清洁，避免污染

### 2. 天贝（Tempeh）

**原理：** 大豆通过根霉菌发酵，形成紧密的块状

**制作要点：**
- 需要购买天贝菌种
- 大豆需要去壳、煮熟
- 在温暖潮湿处发酵24-48小时
- 产生坚果味和紧实质地

## 自制酱料和调味品

### 1. 素蚝油（Vegan Oyster Sauce）

```
材料：
- 1杯香菇（干）
- 2杯水
- 2汤匙酱油
- 1汤匙糖
- 1汤匙玉米淀粉
- 1茶匙芝麻油

步骤：
1. 香菇泡发，保留泡发水
2. 香菇和泡发水煮30分钟
3. 过滤，取汤汁
4. 加入其他调料，小火煮至浓稠
```

### 2. 素鱼露（Vegan Fish Sauce）

```
材料：
- 1杯海带（干）
- 2杯水
- 1/4杯酱油
- 1汤匙盐
- 可选：1茶匙蘑菇粉

步骤：
1. 海带泡发，煮30分钟
2. 过滤，取汤汁
3. 加入其他调料
4. 可以进一步浓缩
```

### 3. 哈里萨（Harissa）辣酱

```
材料：
- 干辣椒（去籽）
- 大蒜
- 孜然
- 香菜籽
- 橄榄油
- 盐

步骤：
1. 所有材料搅拌成泥
2. 用橄榄油覆盖储存
```

## 安全注意事项

### 发酵安全

1. **保持清洁**：所有工具和容器必须干净
2. **正确盐浓度**：足够的盐抑制有害细菌
3. **温度控制**：避免过高或过低温度
4. **观察变化**：如有异味、发霉，立即丢弃
5. **密封容器**：使用专用发酵罐或有排气阀的容器

### 常见问题

**发霉了怎么办？**
- 表面少量白霉（好菌）可以刮除
- 如有彩色霉菌，立即丢弃

**太酸了怎么办？**
- 缩短发酵时间
- 降低温度
- 添加更多盐

**不发酵怎么办？**
- 检查温度（太冷或太热）
- 确认菌种是否有效
- 检查盐浓度

## 储存建议

- **发酵食品**：通常需要冷藏保存
- **自制酱料**：使用干净容器，冷藏保存
- **保质期**：根据配方和储存条件，通常1-3个月
- **标签**：标注制作日期和内容

## 小结

自制调味品和发酵食品不仅能提升你的烹饪水平，还能带来独特的健康益处。虽然有些制作过程需要时间和耐心，但结果绝对值得。从简单的泡菜开始，逐步尝试更复杂的发酵食品，你会发现一个全新的风味世界。

> 🥬 "The best flavors come from patience and time."
>
> 最好的风味来自耐心和时间。

记住：**发酵是一门艺术，需要实践和探索**。不要害怕失败，每一次尝试都是学习的机会！
