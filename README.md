# HowToCookVegan

## 项目介绍

HowToCookVegan 是一个开源项目，旨在为所有热爱素食、关注健康与地球的人，提供一份简单、实用、有趣的纯素食烹饪指南。

在这里，你可以找到：

- 从零开始学会做素食的步骤；
- 各种植物性食材的使用技巧；
- 均衡营养的搭配建议；
- 来自世界各地的素食食谱；
- 以及一群热爱生活、分享善意的同伴。

## 我们希望解决的问题

- “我想吃素，但不知道怎么搭配营养？”
- “做素食是不是又贵又麻烦？”
- “我只会炒青菜，能不能做出更好吃的素菜？”
- “想减少肉类摄入，但不想失去美味。”

如果你也有这些疑问，这个项目就是为你准备的。

## 项目理念

> “素食不是牺牲，而是一种创造。”

我们相信：

- 植物食物可以多样、美味、营养；
- 烹饪是表达生活热爱的方式；
- 改变饮食结构，就是为地球做的一点小小善举；
- 每个人都能从厨房开始，过上更温柔的生活。

## 参与贡献

**HowToCookVegan** 是一个开放的社区项目，无论你是素食新手、经验厨师、还是营养专家，都可以参与贡献！

**你可以：**

- 添加一道你最喜欢的素食食谱；
- 分享某种植物性食材的烹饪技巧；
- 纠正文档中的内容；
- 帮助翻译、改进排版或结构；
- 分享你对素食文化的理解。

**参与方式：**

1. **Fork** 本仓库
2. **新增或修改内容**
3. **发起 Pull Request**
4. 等待合并

> 🥕 每一次贡献，都是为地球加一份绿色的力量。

## 愿景

让更多人：

- 学会做素食，吃得健康；
- 了解植物性饮食的科学与美味；
- 用开源的方式传播善意与知识；
- 共同建设一个更环保、更友善的世界。
