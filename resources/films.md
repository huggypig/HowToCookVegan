# 素食相关纪录片推荐

纪录片能更直观地呈现植物性饮食背后的科学、健康、环保与伦理议题。以下按主题分类推荐。

## 健康与营养

| 片名 | 简介 |
|:----:|:----|
| **[《餐叉胜于手术刀》（Forks Over Knives）](https://movie.douban.com/subject/5975318/)** | 基于科学研究，探讨全植物性饮食对慢性病的改善作用。 |
| **[《游戏改变者》（The Game Changers）](https://movie.douban.com/subject/27621716/)** | 记录运动员采用植物饮食后的表现，打破“蛋白质=肉类”的迷思。 |
| **[《什么是健康》（What the Health）](https://movie.douban.com/subject/27000052/)** | 讨论饮食与健康、医疗体系之间的联系，揭示工业化养殖对健康的影响。 |
| **[《改善大脑健康》Better Brain Health](https://movie.douban.com/subject/35748511/)** | 讨论各种高糖、高脂肪的饮食对身体和大脑的影响。 |

## 环保与伦理

| 片名 | 简介 |
|:----:|:----|
| **[《地球公民》（Earthlings）](https://movie.douban.com/subject/2328234/)** | 剖析动物利用产业的真实面貌，讨论动物权利与伦理考量。 |
| **[《奶牛阴谋》（Cowspiracy）](https://movie.douban.com/subject/26101767/)** | 探讨畜牧业对环境的影响，呼吁关注饮食与气候危机关系。 |
| **[《我们每日的面包》（Our Daily Bread）](https://movie.douban.com/subject/1957795/)** | 辗转欧洲各国的食品工厂，展示了日常生活背面的关于食物的世界。 |

## 食物系统

| 片名 | 简介 |
|:----:|:----|
| **[《食品公司》（Food, Inc.）](https://movie.douban.com/subject/3564499/)** | 揭示现代食品工业的运作，关注食品安全与可持续食物系统。 |
| **[《可持续食物》（Sustainable）](https://movie.douban.com/subject/26998197/)** | 记录可持续农业实践，鼓励消费者支持友善土地的食物生产。 |
| **[《种子：不为人知的故事》（Seed: The Untold Story）](https://movie.douban.com/subject/27007261/)** | 强调种子多样性与农业未来的重要性。 |

建议观看时搭配记录笔记或分享心得，与家人、朋友讨论，可以更深入理解植物性饮食的价值与挑战。
