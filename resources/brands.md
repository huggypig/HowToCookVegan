## 如何选择素食产品？

### 查看配料表

选择素食产品时要仔细查看配料表：

| 检查项 | 说明 |
|:------:|:----|
| **避免动物成分** | 确保产品不含动物成分 |
| **了解添加剂** | 了解产品中的添加剂是否安全 |
| **检查认证** | 检查产品是否有素食认证 |

### 选择健康产品

选择健康的素食产品：

| 标准 | 说明 |
|:----:|:----|
| **营养均衡** | 选择营养均衡的产品 |
| **减少加工** | 选择加工程度较低的产品 |
| **天然成分** | 选择天然成分较多的产品 |

### 考虑环保因素

选择环保的素食产品：

| 因素 | 说明 |
|:----:|:----|
| **包装** | 选择包装环保的产品 |
| **运输** | 选择本地或运输距离较短的产品 |
| **可持续性** | 选择可持续生产的产品 |
