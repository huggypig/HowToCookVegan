# 面粉的种类与用途

![](./images/flour-banner.jpg#large)

在人类的饮食文化中，尤其是中国，面食占有举足轻重的地位。从馒头、包子、面条，到披萨、戚风蛋糕、蔬菜馅饼，甚至广东肠粉…… 都离不开对它。然而，不同类型的面粉决定了食物的口感与用途，新手往往感到困惑。

所以今天我们好好学习一下面粉中的门道，理解不同面粉的“性格”，才能让食物既健康又好吃。

## 面粉的基本概念

在中文语境中，“面粉”一词常被用来泛指所有粉状的谷物制品。最常见的“面粉”是小麦粉，也就是由小麦研磨而成的粉末。而其他谷物制成的粉，我们会称为“××粉”，例如粘米粉、糯米粉、玉米淀粉等等。所以在日常生活中，你可以大致将“面粉”等同于“小麦粉”。

### 小麦粉的特点与优势

小麦粉是世界上使用最广泛的谷物粉，它含有独特的 **面筋蛋白（Gluten）**，这是小麦区别于其他谷物的关键特性。

- **主要成分**：淀粉占 70% 以上，蛋白质（面筋）、水分、少量脂肪与矿物质共同决定口感。
- **筋度（Gluten Strength）**：蛋白质越高，筋度越强，成品越有嚼劲与支撑力。
- **灰分**：代表矿物质含量；全麦粉灰分高、色泽深、营养丰富。

小麦粉的优势在于：

- 含有面筋，能形成富有弹性的结构；
- 加工适应性强，可发酵、可烘焙、可蒸煮；
- 味道中性，易于与其他食材融合。

### “筋度”到底是什么？

小麦粉中的“筋度”其实指的是 **面筋含量** 和 **面筋强度**（Gluten Strength）。筋度的区别，<u>主要来自小麦品种中蛋白质含量的差异</u>，以及部分由加工筛选过程决定。

- 高筋粉多来自硬质小麦，低筋粉多来自软质小麦。
- 除了品种，制粉过程中的工艺（如研磨细度、是否漂白、是否混合不同批次小麦）也会影响筋度表现。
  - **漂白处理**会让粉质更松软、更适合烘焙蛋糕；
  - **混合粉**则是把高筋粉和低筋粉按比例混合，制成中筋粉；
  - **研磨细度**高的面粉吸水性不同，也会稍微影响筋度表现。

在面粉工厂中，“筋度”并不是主观评估的，而是通过一系列标准化理化测试来量化的，核心目标是评估小麦粉的**蛋白质含量**与**面筋质量**，即“能不能形成有弹性、有延展性的面团”。

“筋度”主要由小麦粉中两种蛋白质决定：

- **麦谷蛋白**（Glutenin） → 提供弹性（让面团能“回弹”）
- **醇溶蛋白**（Gliadin） → 提供延展性（让面团能“拉长”）

这两种蛋白加水后在揉面过程中交联形成三维网络结构 —— **面筋**（Gluten network）。工厂就是通过检测这个网络的性质来量化“筋度”，从而得出面筋指数（GI）。



### 面筋对素食饮食的意义

面筋由麦谷蛋白与醇溶蛋白组成，遇水与揉面时形成弹性网络。对于素食者，面筋是宝贵的植物蛋白来源：

- 每 100 克高筋面粉可提供约 12 克蛋白质。
- 可以通过“洗面筋”制作素肉、素鸡、辣条。
- 对麸质敏感者需改用无麸质替代粉。

## 面粉的分类与用途

### 按筋度划分

面粉（小麦粉）是制作各种主食（面包、饺子、包子、蛋糕）的基础原料。根据蛋白质（面筋）含量的不同，又分为高筋、中筋、低筋三类。

| 类型 | 蛋白质含量 | 典型料理 | 口感特点与技巧 |
|:----:|:----:|:----|:----|
| **高筋面粉** | 11%–14% | 面包、馒头、拉面、披萨 | 筋度强，适合长时间发酵；和风馒头可与 10% 全麦粉混合提升香气 |
| **中筋面粉** | 9%–11% | 饺子、包子、馄饨皮、家常面 | 兼顾柔软与筋性；包子皮可加 5% 油脂，蒸后更柔软 |
| **低筋面粉** | 7%–9% | 蛋糕、饼干、挞皮、炸浆 | 筋度低、口感松软；可与玉米淀粉混合做酥脆炸浆 |

### 按原料 / 用途细分

- **全麦粉（Whole Wheat Flour）**：保留胚芽与麸皮，膳食纤维较高，适合做杂粮馒头、全麦吐司。建议与高筋粉 1:1 混合，口感更柔软。
- **特制粉**：（属于小麦粉的再调配产品，只是根据用途调整蛋白质、灰分或添加膨松剂）
  - **饺子粉 / 馒头粉**：以中高筋小麦粉为基础，额外筛选细度与稳定吸水率，能保持饺子皮韧性。
  - **蛋糕粉 / 面包粉**：本质仍是小麦粉，厂商通过混合不同批次的粉提升一致性（无额外香精）。
  - **自发粉**：在低筋小麦粉中加入泡打粉与少量盐，使用时不必另加膨松剂。
- **高粉＋低粉混合**：例如自制中筋粉 ≈ 高筋粉 70% + 低筋粉 30%。

### 米粉：来自稻米的“非面筋粉”

与小麦粉不同，**粘米粉**（Rice Flour）和 **糯米粉**（Glutinous Rice Flour） 都是以稻米为原料研磨而成的。它们的共同点是 —— **不含面筋（无麸质）**，因此揉不出“面团”，口感更柔软或Q弹。

| 名称                                    | 原料         | 特点               | 常见用途               |
| :-------------------------------------- | ------------ | ------------------ | ---------------------- |
| 粘米粉（Rice Flour）                    | 普通大米     | 粒细色白，略有黏性 | 肠粉、米皮、米饼、凉糕 |
| 糯米粉（Glutinous Rice Flour）          | 糯米         | 粘性强、口感软糯   | 汤圆、糍粑、麻薯       |
| 熟糯米粉（Cooked Glutinous Rice Flour） | 糯米炒熟研磨 | 不吸水、香气浓郁   | 冰皮月饼、糕点外皮     |
| 米浆粉（Rice Batter Mix）               | 粘米粉＋木薯粉 | 调校好比例，蒸后晶莹 | 粤式肠粉、萝卜糕、素虾饺 |

:::tip

糯米粉叫 *Glutinous Rice Flour*（看似“有面筋”），其实是误译。这里的 “glutinous” 指“粘稠”，而不是“含面筋”。所以糯米粉是完全无麸质的，适合对面筋过敏的素食者。

:::

### 淀粉粉类：为口感加分的“隐形配角”

除了谷物粉，厨房常见的淀粉（Starch）也非常实用：

|              名称              | 来 源    | 主要用途                         | 小技巧                                              |
| :----------------------------: | :------- | :------------------------------- | :-------------------------------------------------- |
|  **玉米淀粉（Corn Starch）**   | 玉米胚乳 | 勾芡、炸衣、烘焙中提升松化       | 勾芡时需先与冷水调匀；做炸物时与低筋粉 1:1 可更酥脆 |
|    **太白粉（马铃薯淀粉）**    | 马铃薯   | 勾芡、调制素凉粉、珍珠丸子外层   | 遇热即稠，适合最后阶段增稠                          |
|          **红薯淀粉**          | 红薯     | 拉皮、粉条、凉粉                 | 口感弹韧，可与玉米淀粉混合做素鱿鱼圈                |
| **木薯淀粉（Tapioca Starch）** | 木薯根   | 珍珠粉圆、巴西芝士包、无麸质面包 | 提供 Q 弹口感，但单用会偏黏                         |

玉米淀粉除了勾芡，还能：

1. **制作素汤厚度**：如玉米浓汤、蘑菇汤，勾芡后更丝滑。
2. **增加炸物酥脆感**：包裹豆腐、茄子或蘑菇炸制，外皮薄脆。
3. **防止面团粘连**：撒在烘焙台或模具中，替代小麦粉避免过度增加筋度。
4. **做素奶酪酱**：与植物奶、营养酵母煮至浓稠，口感类似芝士酱。

## 如何选择面粉

| 料理 | 推荐面粉 | 选择技巧 |
|:----:|:----|:----|
| 面包 / 披萨 | 高筋 | 选“强筋”或“高筋”标注，颜色偏乳白，做吐司可加 1% 橄榄油减少干裂 |
| 饺子 / 馒头 | 中筋 | 看吸水率（≥58% 更耐揉），冷冻饺子皮可选略高筋度防破皮 |
| 面条 / 拉面 | 中高筋 | 可加少量碱水、盐提高韧性 |
| 蛋糕 / 曲奇 | 低筋 | 选未漂白、蛋糕专用粉；做戚风时筛粉两遍，避免颗粒 |
| 广式肠粉 / 米皮 | 粘米粉或复合粉 | 不含面筋，需搭配木薯粉增加柔韧 |

> 专业建议：阅读包装的“蛋白质”、“灰分”、“吸水率”标注，比仅看“高/中/低筋”更准确。

## 无麸质与替代粉

对麸质不耐或想尝试不同风味时，可善用下列替代粉：

| 替代粉 | 营养亮点 | 料理应用 | 注意事项 |
|:----:|:----|:----|:----|
| **糙米粉** | 无麸质、富维生素 B、矿物质 | 中式糕点、米饼、可与木薯粉混合做肠粉 | 易沉淀，使用前需搅拌均匀 |
| **燕麦粉** | β-葡聚糖，帮助胆固醇管理 | 早餐松饼、能量棒、燕麦曲奇 | 需确认“无麸质认证”以避免交叉污染 |
| **鹰嘴豆粉** | 蛋白质、铁、叶酸丰富 | 印度煎饼、蔬菜咸派底、素炸物挂糊 | 烹煮前要炒熟去豆腥 |
| **木薯粉 / 葛根粉** | 无味、黏性强 | 提升烘焙湿润度、做珍珠粉圆 | 不宜大量使用，口感会偏黏 |
| **藜麦粉** | 完整蛋白质、膳食纤维 | 无麸质面包、意式烤饼 | 口味较浓，可与米粉 1:1 调和 |
| **荞麦粉** | 升糖指数低，含芦丁（Rutin）护血管 | 日式荞麦面、俄式煎饼 Blini、荞麦饼干 | 不含麸质，面团需搭配木薯粉或小麦粉成型 |
| **小米粉 / 高粱粉** | 富含 B 族及铁质 | 北方杂粮饼、粥底、蒸糕 | 含油脂较多，易氧化，开封后冷藏 |
| **绿豆粉 / 豌豆粉** | 植物蛋白、淀粉双高 | 绿豆凉粉、素肉松、素丸子黏合 | 建议与其他粉类调和，避免粉质过硬 |



## 面粉与营养平衡

- **蛋白质**：高筋粉≒12g/100g；与豆制品、坚果搭配能提升膳食品质。
- **膳食纤维**：全麦粉约 10g/100g，是白面粉的 4 倍，有助肠道健康。
- **微量元素**：全麦粉含镁、锌、铁；精制粉则损失较多。
- **血糖负荷**：白面粉升糖快，搭配蔬菜、蛋白质或选择杂粮粉可降低餐后血糖波动。

> 健康提示：避免长期过度精制粉，餐桌上可以“白面粉 50% + 全麦 / 豆类或杂粮粉 50%”的方式搭配，既保留口感又提升营养。

## 制作与操作技巧

1. **吸水率差异**：每批面粉的吸水性不同，和面时可先保留 10% 水量，再逐步加入。
2. **混合技巧**：想要柔软又有营养，可加入 5% 油脂、10% 糯米粉或马铃薯泥。
3. **醒面时间**：高筋粉需 30 分钟以上醒发，中筋粉约 20 分钟，低筋粉多用于烘焙，无需醒面。
4. **无麸质烘焙**：缺乏筋性时，可加入洋车前子壳粉、亚麻籽粉、木薯淀粉提升结构。
5. **粉类混搭图表**：
   - 想要“Q 弹”：小麦粉 80% + 木薯淀粉 20%（适合素拉面、手擀面）。
   - 想要“松软”：小麦粉 70% + 玉米淀粉 30%（适合饼干、炸浆）。
   - 想要“无麸质但结构稳定”：米粉 50% + 荞麦粉 30% + 木薯淀粉 20%。

## 面粉保存与安全

- **储存环境**：阴凉、干燥、密封；潮湿环境易长虫、发霉。
- **保存期限**：精制粉约 6–12 个月；全麦粉因含油脂，开封后建议冷藏并于 3 个月内用完。
- **识别变质**：若闻到酸败味、出现结块或小虫，应立即丢弃。
- **优先选择**：未漂白、无铝泡打粉、非转基因认证的面粉，对身体更友善。

## 厨房小结

- 拥有至少两种面粉（高筋 + 低筋或高筋 + 全麦）就能应对绝大多数植物料理。
- 素食烘焙可利用面粉建立主体，再融合植物奶油、坚果酱、生酮甜味剂，创造层次。
- 透过记录不同品牌、吸水率与操作体验，建立自己的“面粉笔记本”，是成为面点高手的捷径。

无论是传统手擀面还是创新的无麸质烘焙，只要理解面粉的性格，素食厨房也能随心所欲、百变精彩。让面粉成为日常膳食中温暖又可靠的伙伴吧！
