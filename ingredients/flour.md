# 面粉的种类与用途

在人类的饮食文化中，尤其是中国，面食占有举足轻重的地位。面包、饺子、面条、馒头、蛋糕、年糕、肠粉…… 都离不开对面粉。然而，不同类型的面粉决定了食物的口感与用途，新手往往感到困惑。所以今天我们好好学习一下面粉中的门道，从松软的面包到劲道的面条，都离不开对面粉的正确选择与理解。



## 面粉的基本概念

小麦粉的主要成分：**淀粉、蛋白质（面筋）、水分、少量脂肪与矿物质**

- 面粉的来源：小麦研磨
- “筋度”与蛋白质含量的关系
- 面粉中淀粉、蛋白质、灰分的作用



## 面粉的分类与用途

### 按筋度分类

| 类型                         | 蛋白质含量 | 典型用途           | 特点                     |
| ---------------------------- | ---------- | ------------------ | ------------------------ |
| 高筋面粉 (Bread Flour)       | 11%–14%    | 面包、馒头、面条   | 筋度强，适合发酵、嚼劲足 |
| 中筋面粉 (All-purpose Flour) | 9%–11%     | 包子、饺子、馄饨皮 | 兼顾柔软与筋性，最通用   |
| 低筋面粉 (Cake Flour)        | 7%–9%      | 蛋糕、饼干、蛋挞皮 | 筋度低、口感松软酥脆     |



### 按原料和用途分类

- **全麦粉（Whole Wheat Flour）**：保留胚芽和麸皮，营养更高，颜色偏深。
- **高粉＋低粉混合**：可自由调节筋度，例如中筋≈高粉 × 70% + 低粉 × 30%。
- **特制粉**：
  - **饺子粉 / 馒头粉**：中高筋，稳定筋度。
  - **蛋糕粉 / 面包粉**：工业标准化产品，适合初学者。
  - **自发粉（Self-raising Flour）**：含泡打粉，不需再加酵母。



## 如何选择面粉

| 食物        | 推荐面粉类型               | 原因                   |
| ----------- | -------------------------- | ---------------------- |
| 面包        | 高筋                       | 需要强筋度形成支撑结构 |
| 饺子 / 馒头 | 中筋                       | 筋度适中，柔韧不粘     |
| 面条        | 中高筋                     | 保持弹性与嚼劲         |
| 蛋糕 / 曲奇 | 低筋                       | 口感松软酥脆           |
| 肠粉 / 米皮 | 非小麦粉（粘米粉、淀粉类） | 不含筋性               |



## 非小麦粉与替代粉

介绍素食者常用的“无麸质替代粉”：

- **糙米粉**：轻盈、无麸质。
- **鹰嘴豆粉**：富含蛋白质，可做煎饼。
- **木薯粉 / 玉米淀粉**：改善口感，常用于素炸物。
- **燕麦粉**：健康低脂，适合早餐烘焙。



## 面粉保存建议

- 储存环境：阴凉、干燥、密封
- 如何辨别面粉变质
- 选择未漂白、无添加面粉更健康
