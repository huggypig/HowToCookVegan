# 常备食材清单

正所谓“巧妇难为无米之炊”，储备充足的食品、食材是出色素食烹饪的基础。花点时间，建立合理的素食食品柜，不仅能让你随时做出美味的素食菜肴，还能节省时间和金钱。这份清单将帮助你建立一个实用的素食厨房。

## 基础食材

### 谷物类

| 食材 | 类型 | 特点/用途 |
|:----:|:---:|:--------:|
| **大米** | 必备 | 白米、糙米（至少一种） |
| **全麦面粉** | 必备 | 用于烘焙和面食 |
| **燕麦** | 必备 | 传统燕麦或即食燕麦 |
| **挂面或方便面** | 推荐 | 快速方便，应对突发情况 |
| **藜麦** | 推荐 | 完整蛋白质 |
| **小米** | 推荐 | 易消化 |
| **全麦面条** | 推荐 | 方便快捷 |

### 豆类

| 食材 | 类型 | 特点/用途 |
|:----:|:---:|:--------:|
| **扁豆** | 必备 | 红扁豆、绿扁豆（不需要浸泡，快速） |
| **黑豆** | 必备 | 用途广泛 |
| **鹰嘴豆** | 必备 | 可以做鹰嘴豆泥 |
| **黄豆、绿豆、红豆** | 推荐 | 基础豆类，应对突发情况 |
| **各种豆类罐头** | 推荐 | 方便快捷（虽然营养略少） |

### 豆制品

| 食材 | 类型 | 特点/用途 |
|:----:|:---:|:--------:|
| **豆腐** | 必备 | 嫩豆腐、老豆腐（至少一种） |
| **豆干** | 必备 | 质地紧实，适合炒菜 |
| **豆浆** | 推荐 | 可以直接饮用或烹饪 |
| **腐竹** | 推荐 | 传统食材 |

### 蔬菜类

**新鲜蔬菜（根据季节选择）：**
- **叶菜**：菠菜、生菜、小白菜
- **根茎**：胡萝卜、土豆、白萝卜
- **果菜**：番茄、茄子、青椒、黄瓜
- **菌类**：香菇、平菇、杏鲍菇

**常备蔬菜（应对突发情况）：**
- **豆腐**：蛋白质来源，用途广泛
- **青椒、胡萝卜、黄瓜、西红柿**：基础配菜
- **木耳**：口感独特，可长期储存
- **茄子**：适合多种烹饪方式

**干货（可以长期储存）：**
- **干香菇**：鲜味来源
- **海带/紫菜**：鲜味和矿物质
- **木耳**：口感独特

### 坚果和种子

| 食材 | 类型 | 特点/用途 |
|:----:|:---:|:--------:|
| **杏仁** | 必备 | 用途广泛 |
| **腰果** | 必备 | 制作素奶油 |
| **芝麻** | 必备 | 白芝麻、黑芝麻 |
| **核桃** | 推荐 | Omega-3 来源，应对突发情况 |
| **奇亚籽/亚麻籽** | 推荐 | Omega-3 和鸡蛋替代 |
| **南瓜籽** | 推荐 | 营养丰富 |

## 调味品

### 基础调味

**高频使用的调味品（建议常备）：**

| 食材 | 类型 | 特点/用途 |
|:----:|:---:|:--------:|
| **盐** | 必备 | 海盐或精制盐 |
| **糖** | 必备 | 白砂糖、冰糖、棉白糖（至少一种） |
| **松茸鲜/素鲜粉** | 可选 | 纯植物来源的增鲜调料（鸡精替代品） |
| **黑胡椒、白胡椒** | 必备 | 整粒或粉 |
| **椰糖** | 推荐 | 天然甜味剂 |
| **枫糖浆** | 推荐 | 液体甜味剂 |

### 酸味

**高频使用的酸味调料（建议常备）：**

| 食材 | 类型 | 特点/用途 |
|:----:|:---:|:--------:|
| **米醋** | 必备 | 或白醋 |
| **黑醋（香醋、陈醋）** | 必备 | 中式料理 |
| **白醋** | 必备 | 清爽酸味 |
| **柠檬** | 必备 | 新鲜或柠檬汁 |
| **苹果醋** | 推荐 | 有果香 |

### 鲜味

**高频使用的鲜味调料（建议常备）：**

| 食材 | 类型 | 特点/用途 |
|:----:|:---:|:--------:|
| **生抽** | 必备 | 基础调味 |
| **老抽** | 必备 | 上色用 |
| **素耗油（蘑菇耗油）** | 必备 | 以蘑菇、菌菇为原料的增鲜调料 |
| **料酒（黄酒）** | 可选 | 去腥、增香 |
| **豆瓣酱** | 推荐 | 川菜必备 |
| **味噌** | 推荐 | 发酵鲜味 |
| **营养酵母** | 推荐 | "奶酪"味 |

> **温馨提示：** 选购增鲜类调味品时，请检查配料表，确认不含鱼露、贝类提取物等动物成分；有「纯素」「Plant-based」等标识会更安心。

### 香料

**高频使用的香料（建议常备）：**

| 类别 | 食材 | 特点/用途 |
|:----:|:----|:---------|
| **中式基础** | 大葱、小葱 | 增香、调味（纯净素人群可用香菜、香椿等替代） |
| | 生姜、大蒜 | 去腥、增香（如遵循去五辛，可改用姜黄、香茅等） |
| | 花椒 | 麻味、增香 |
| | 八角、桂皮、香叶 | 卤味、炖菜 |
| | 干辣椒、小米椒 | 辣味（根据喜好） |
| | 五香粉（或十三香） | 复合香料 |
| **通用** | 黑胡椒、白胡椒 | 基础调味 |
| **推荐** | 姜黄 | 抗炎、上色 |
| | 小茴香 | 温暖、泥土味 |
| | 肉桂 | 甜、温暖 |

### 油类

| 食材 | 类型 | 特点/用途 |
|:----:|:----|:---------|
| **菜籽油/葵花籽油** | 必备 | 高温烹饪 |
| **橄榄油** | 必备 | 特级初榨（生食）、精炼（烹饪） |
| **芝麻油** | 推荐 | 调味 |
| **椰子油** | 推荐 | 烘焙 |
| **藤椒油** | 推荐 | 麻香鲜爽，适合凉拌、蘸汁及川味热菜 |
| **木姜子油** | 推荐 | 柑橘香气，清香辛辣，适合拌菜、热拌绿叶菜 |

## 建立食品柜的步骤

不需要一开始就很完美，我们可以从基础开始，逐步扩展，再进行个性化打造。

| 第一阶段                                                     | 第二阶段                                                     | 第三阶段                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| <ul><li>基础谷物（大米、面粉）</li><li>基础豆类（扁豆、黑豆）</li><li>基础调味（盐、糖、酱油、醋）</li><li>基础油（菜籽油、橄榄油）</li></ul> | <ul><li>添加更多豆类</li><li>添加坚果和种子</li><li>添加更多香料</li><li>添加发酵食品</li></ul> | <ul><li>根据个人喜好添加</li><li>根据常用菜系添加</li><li>添加特殊食材</li></ul> |



## 储存建议

#### 1. 分类储存

**干燥食材：**
- 谷物、豆类、面粉
- 密封容器，防虫防潮

**调味品：**
- 避光、阴凉
- 使用密封容器

**油类：**

- 避光、阴凉
- 某些油（如亚麻籽油）需要冷藏

#### 2. 标签和日期

**建议：**
- 标注购买日期
- 标注过期日期
- 使用"先进先出"原则

#### 3. 定期检查

**建议：**
- 定期检查库存
- 及时补充消耗的食材
- 清理过期食材



## 常见问题

#### 1. 需要准备多少？

**答案：**
- 根据家庭大小和个人习惯
- 建议：至少能支持一周的烹饪

#### 2. 需要购买替代品吗？

**答案：**
- 不是必须
- 可以根据个人喜好和预算选择
- 传统食材也能做出美味素食

#### 3. 如何避免浪费？

**答案：**
- 合理规划
- 使用"先进先出"原则
- 定期检查库存

## 厨房小结

建立合理的素食食品柜，是素食烹饪的基础。通过逐步积累，合理规划，你就能随时做出美味的素食菜肴。

> 🥫 "A well-stocked pantry is like a well-stocked toolbox - you have what you need when you need it."
>
> 储备充足的食品柜就像储备充足的工具箱——当你需要时，你就有你需要的。

记住：**从基础开始，逐步扩展**，根据个人喜好和需求建立属于你的素食食品柜！
