# 热菜

热菜是素食餐桌上的主角，这里收录了各种美味的素食热菜食谱。

## 食谱特点

本部分的热菜食谱包括：

- **经典川菜**：如宫保豆腐、麻婆豆腐等
- **家常炒菜**：如红烧茄子、香菇炒青菜等
- **地方特色**：如地三鲜、素狮子头等

## 烹饪技巧

制作热菜时需要注意：

- **火候控制**：不同食材需要不同的火候，注意控制
- **调味平衡**：掌握好各种调料的用量，保持味道平衡
- **食材处理**：提前处理好食材，可以提高制作效率

## 推荐食谱

- **宫保豆腐**：经典的素食川菜，口感鲜香微辣
- **红烧茄子**：经典的家常菜，色泽红亮，口感软糯
- **麻婆豆腐**：经典的川菜，麻辣鲜香，下饭神器

开始探索这些美味的素食热菜吧！
