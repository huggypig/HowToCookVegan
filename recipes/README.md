# 素食食谱

欢迎来到素食食谱部分！这里收录了各种美味的素食食谱，从简单易做的家常菜到精致的素食料理，应有尽有。

## 食谱分类

本部分按照菜品种类分为以下几个类别：

- **热菜**：各种炒菜、炖菜、蒸菜等热菜食谱
- **凉菜**：清爽开胃的凉菜食谱
- **汤品**：营养丰富的汤品食谱
- **主食**：米饭、面条、饺子等主食食谱
- **糕点**：各种甜点和糕点食谱
- **饮料**：植物奶、果汁等饮品食谱

## 使用建议

- **选择适合的难度**：每个食谱都标注了预估烹饪难度，从 ★ 到 ★★★★，可以根据自己的烹饪水平选择
- **准备食材**：按照食谱中的"计算"部分准备食材，确保分量准确
- **遵循步骤**：按照"操作"部分的步骤逐步制作，注意火候和时间的控制
- **参考附加内容**：每个食谱的"附加内容"部分提供了实用的技巧和建议

## 开始制作

选择您感兴趣的食谱，准备食材，开始您的素食烹饪之旅吧！

如果您有任何问题或建议，欢迎提出 Issue 或 Pull request。
