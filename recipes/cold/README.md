# 凉菜

凉菜是素食餐桌上的清爽之选，适合夏季食用，也适合作为开胃小菜。

## 食谱特点

本部分的凉菜食谱包括：

- **清爽开胃**：如凉拌木耳、拍黄瓜等
- **营养丰富**：如豆腐皮卷等
- **制作简单**：大多数凉菜制作简单，适合初学者

## 制作技巧

制作凉菜时需要注意：

- **食材新鲜**：选择新鲜的食材，保证口感和营养
- **调味平衡**：掌握好各种调料的用量，保持味道平衡
- **保存方法**：了解凉菜的保存方法，延长保质期

## 推荐食谱

- **凉拌木耳**：清爽开胃，营养丰富
- **拍黄瓜**：经典的凉菜，制作简单
- **豆腐皮卷**：口感丰富，营养均衡
