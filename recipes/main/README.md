# 主食

主食是素食餐桌上的基础，这里收录了各种美味的素食主食食谱。

## 食谱特点

本部分的主食食谱包括：

- **营养均衡**：如素炒饭、什锦炒面等
- **制作简单**：大多数主食制作简单，适合日常食用
- **口感丰富**：如素饺子等

## 制作技巧

制作主食时需要注意：

- **米饭处理**：使用隔夜饭或冷却的米饭，可以让炒饭更松散
- **面条处理**：煮制面条时注意时间，避免过软或过硬
- **馅料处理**：制作饺子时注意馅料的处理，避免出水

## 推荐食谱

- **素炒饭**：简单美味，营养丰富
- **什锦炒面**：口感丰富，制作简单
- **素饺子**：传统主食，可以提前制作
