# 学会看配料表和营养成分表

> "Knowledge is power - reading labels empowers you to make informed choices." —— 知识就是力量，学会阅读标签让你做出明智的选择。

食品包装正面五彩斑斓，但真正决定产品成分与安全性的，却藏在背面的配料表与营养成分表。学会阅读食品标签和配料表，是素食者必备的技能。它能帮助你识别隐藏的动物成分，选择更健康的食品，做出明智的购买决定。



## 读配料表的三步法

只要掌握以下步骤，你就能快速判断一件食品是否适合纯素饮食、是否含有过多添加剂，以及它与健康目标（控糖、低油、低钠）是否匹配。

1. **确认列出位置**：通常位于包装背面或侧面，常见标题为 “配料” 或 “Ingredients”。
2. **观察排序规则**：成分按照含量从高到低排列，排在前几位的就是你吃下最多的内容。
3. **寻找“红旗”关键词**：动物来源、含糖、反式脂肪、人工添加剂等，是素食者与健康饮食需要特别注意的部分。

## 快速判断是否纯素

- **先看认证标志**：Vegan Society、V-Label、PETA 等认证能节省大量判断时间。
- **扫描关键词**：蜂蜜、乳粉、黄油、明胶、鱼胶、蚝油、奶油等都属于动物来源。
- **留意 “May Contain”**：表示生产线可能接触过某种过敏原；对纯素者一般可接受，但对严重过敏者需谨慎。
- **配料少且天然最好**：成分越少越透明，越容易识别风险。

## 需要警惕的动物成分

### 1. 明显的动物来源

| 类别 | 常见标注 | 提示 |
|:----:|:----|:----|
| 肉类 | 牛肉、猪肉、鸡肉、鱼肉、海鲜、肉松 | 直接非素食 |
| 乳制品 | 牛奶、奶油、奶酪、黄油、乳清、乳粉 | 乳粉常隐藏在咖啡伴侣、饼干中 |
| 蛋类 | 鸡蛋、蛋清、蛋黄、蛋白粉、蛋液 | 烘焙/面食常见 |
| 蜂产品 | 蜂蜜、蜂蜡、蜂王浆、花粉 | 纯素者需避免 |

### 2. 隐藏的动物成分

| 成分 | 来源 | 常见产品 | 备注 |
|:----:|:----|:----|:----|
| 明胶 | 动物皮、骨 | 果冻、糖果、酸奶、胶囊 | 纯素可用琼脂、果胶替代 |
| 凝乳酶 | 小牛胃 | 某些奶酪 | 市面上有微生物凝乳酶可替代 |
| 胭脂虫红 | 昆虫萃取 | 饮料、糖果、口红 | 又称胭脂红、红色 4 号、Cochineal |
| L-半胱氨酸 | 鸡羽毛或人/动物毛发 | 面包改良剂 | 注意 “L-Cysteine” 或 “E920” |
| 维生素 D3 | 羊毛脂 | 强化植物奶、谷物 | 改选维生素 D2 或藻油 D3 |
| Omega-3 | 鱼油 | 健康油、补充剂 | 留意是否标注 “海藻油” |

## 食品添加剂解析

当配料表出现 “E120”、“E441” 之类的编号或陌生名称，不要立刻恐慌，它们是食品添加剂的统一编码（欧盟 E 编号体系）。关键在于识别其来源与用途，并评估是否适合你的饮食原则。

### 1. 常见含动物成分或争议添加剂

| 编号/名称 | 功能 | 主要来源 | 纯素提示 |
|:----:|:----|:----|:----|
| E120 胭脂虫红 | 着色剂 | 胭脂虫 | 改选植物性色素（甜菜红、辣椒红） |
| E441 明胶 | 凝胶剂 | 动物骨皮 | 改用琼脂、果胶、卡拉胶 |
| E542 骨磷酸盐 | 抗结剂 | 动物骨头 | 咖啡伴侣、粉状食品常见 |
| E631/E635 | 增味剂 | 鱼、肉或酵母 | 选标注 “酵母抽取物” 的产品 |
| E904 虫胶 | 上光剂 | 紫胶虫分泌物 | 多见于糖衣、亮面水果 |
| E910/E920 半胱氨酸 | 面粉改良剂 | 羽毛、毛发 | 部分厂商使用玉米发酵来源，需确认 |

### 2. 常见人工/复配添加剂

| 添加剂 | 类型 | 常见用途 | 留意事项 |
|:----:|:----|:----|:----|
| 起酥油（Shortening） | 人造固体脂肪 | 烘焙、酥皮、饼干 | 可能含氢化油或反式脂肪，选 “零反式脂肪” 或植物油版本 |
| 人造奶油（Margarine） | 植物油 + 乳化剂 | 烘焙、涂抹酱 | 留意是否含乳固体及反式脂肪 |
| 乳粉 / 奶精粉 | 乳制品 | 咖啡伴侣、奶茶粉 | 直接非纯素；有椰奶粉、燕麦奶粉替代 |
| 食用香精 | 调味剂 | 饮料、糖果、烘焙 | “天然香精” 也可能来自动物，建议咨询厂商 |
| 甜蜜素（Cyclamate） | 人工甜味剂 | 无糖饮料、蜜饯 | 建议控制摄入，孕妇慎用 |
| 安赛蜜（Acesulfame K） | 人工甜味剂 | 糖果、烘焙 | 常与阿斯巴甜搭配，儿童减少摄入 |
| 阿斯巴甜（Aspartame） | 人工甜味剂 | 无糖饮料、口香糖 | 苯丙酮尿症患者禁用 |
| 山梨酸钾 / 苯甲酸钠 | 防腐剂 | 酱料、果酱、饮料 | 国家标准允许剂量，仍需控制总摄入 |
| 亚硝酸盐 | 防腐剂、护色剂 | 腌制品 | 长期过量会增加健康风险 |
| 乳化剂（单甘油酯、双甘油酯） | 乳化稳定 | 烘焙、冰淇淋 | 原料可来自植物油或动物脂肪，需确认 |
| 食用香料（香兰素等） | 天然/合成香味 | 饼干、蛋糕 | 留意是否含溶剂、是否标注植物来源 |



### 3. 如何评估添加剂对健康的影响

1. **看份量**：单项添加剂在法规范围内通常安全，但多种叠加时需关注总摄入量。
2. **读备注**：配料表后常有 “含有抗氧化剂（维生素 C）” 等提示，有助判断用途。
3. **整合营养成分表**：添加剂多的产品往往钠、糖、脂肪也偏高，需要同时关注营养成分表。

## 营养成分表：量化健康指标

| 指标 | 关注重点 | 纯素饮食建议 |
|:----:|:----|:----|
| 能量 | 每份 vs 每100克 | 控制热量时先看每份，比较产品时看每100克 |
| 脂肪/饱和脂肪 | 过量会影响心血管 | 优先选择饱和脂肪 < 5g/100g 的产品 |
| 反式脂肪 | 应为 0 | 起酥油、人造奶油使用产品需特别留意 |
| 钠 | < 600mg/100g | 高血压或控盐人群尤其重要 |
| 糖 | < 5g/100g | 区分天然糖（水果）与添加糖；关注 “含糖量占能量百分比” |
| 纤维 | ≥ 3g/100g | 高纤维食品有助血糖、肠道健康 |

## 实用检查清单

- [ ] 主要配料是否为全谷物、豆类、坚果、蔬果等天然食材？
- [ ] 是否含有明确的动物来源或可能的动物添加剂？
- [ ] 添加剂数量是否过多（超过 6-8 项需更谨慎）？
- [ ] 营养成分表中的钠、糖、脂肪是否符合健康目标？
- [ ] 是否有可信的纯素认证或品牌透明度高？

## 常见误区与陷阱

1. **“天然” 不等于 “纯素”**：天然香料、天然色素仍可能来自动物。
2. **“植物性” 不等于 “纯素”**：某些植物奶仍添加乳粉或蜂蜜，需要确认配料。
3. **配方随时可能变**：曾经的素食产品后来加入动物成分，因此每次购买都要复查配料。
4. **营养强化需辨别来源**：DHA、维生素 D、钙强化等要确认是否来源于海藻或植物。

## 厨房小结

- 读配料表的顺序：先锚定主材料 → 排除动物成分 → 评估添加剂 → 核对营养成分。
- 充分利用数字工具：条码扫描 App、国家食品安全信息平台、品牌客服都是可靠伙伴。
- 让阅读标签成为习惯，才能在琳琅满目的商超货架中，快速找到真正适合你的素食好物。

记住：**仔细阅读，持续学习**，让标签成为你的朋友，而不是敌人！
