# 学会看配料表

> "Knowledge is power - reading labels empowers you to make informed choices." —— 知识就是力量——阅读标签让你做出明智的选择。

学会阅读食品标签和配料表，是素食者必备的技能。它能帮助你识别隐藏的动物成分，选择更健康的食品，做出明智的购买决定。

## 配料表基础

### 1. 配料表的位置

**通常位置：**
- 包装背面或侧面
- 在营养成分表附近
- 通常以"配料"或"Ingredients"开头

### 2. 配料表的排序规则

**重要原则：**
- **按含量从高到低排列**
- 含量最高的配料排在第一位
- 含量最低的排在最后

**实际应用：**
- 如果糖排在前面，说明含糖量高
- 如果全谷物排在前面，说明是健康选择

## 需要警惕的动物成分

### 1. 明显的动物成分

| 类别 | 成分 | 说明 |
|:----:|:----|:----|
| **肉类** | 牛肉、猪肉、鸡肉、鱼肉等 | 通常不会出现在素食产品中 |
| **乳制品** | 牛奶、奶油、奶酪、黄油、乳清蛋白、乳清粉、酪蛋白、酪蛋白酸钠 | - |
| **蛋类** | 鸡蛋、蛋清、蛋黄、蛋粉 | - |
| **蜂蜜** | 蜂蜜、蜂蜡、蜂王浆 | 纯素者需要避免 |

### 2. 隐藏的动物成分

| 成分 | 来源 | 常见于 | 备注 |
|:----:|:----|:------|:----|
| **明胶** | 动物骨骼、皮肤 | 果冻、软糖、某些酸奶、胶囊 | - |
| **凝乳酶** | 小牛胃 | 某些奶酪 | 需要特别检查 |
| **胭脂虫红** | 昆虫 | 某些红色食品、化妆品 | - |
| **角叉菜胶** | 海藻 | - | 虽然来自海藻，但某些纯素者避免使用（可能有健康争议） |
| **L-半胱氨酸** | 可能是动物毛发或羽毛 | 某些面包 | - |
| **维生素D3** | 可能是羊毛脂（羊毛） | 强化食品 | 纯素者应该选择维生素D2或纯素D3 |
| **Omega-3** | 鱼油（非纯素） | 某些补充剂 | 纯素来源：亚麻籽、奇亚籽、海藻 |

### 3. 可能含有动物成分的添加剂

| 添加剂 | 来源 | 说明 |
|:------:|:----|:----|
| **E120** | 胭脂虫红（昆虫） | - |
| **E441** | 明胶（动物） | - |
| **E542** | 骨粉（动物） | - |
| **E904** | 虫胶（昆虫） | - |
| **乳清** | 来自牛奶 | - |
| **酪蛋白** | 来自牛奶 | - |
| **乳糖** | 来自牛奶 | - |

在配料表里，你可能会看到“E120”“E441”这类以字母 E 开头的编号。**E 编号**是欧盟及多国通用的食品添加剂代码体系，用来简化添加剂名称。每个编号对应一种特定功能（着色、保鲜、增味等），但编号本身并不会告诉你来源是否为动物，因此素食者需要自行查证来源。[健康指南文章参考指出，多数常见的非素食 E 编号包括 E120（胭脂虫红）、E441（明胶）、E542（骨粉）、E631/E635（增味剂，常以鱼或肉制成）、E904（虫胶）、E910/E920（半胱氨酸，常来源于羽毛或毛发）](https://www.healthyd.com/articles/expert-blog/%E5%88%86%E8%BE%A8%E7%87%9F%E9%A4%8A%E6%A8%99%E7%B1%A4-%E9%9D%9E%E7%B4%A0%E9%A3%9F-e%E7%B7%A8%E8%99%9F-%E9%A3%9F%E5%93%81%E6%B7%BB%E5%8A%A0%E5%8A%91)。遇到陌生编号时，可：

1. 查阅可靠的食品添加剂数据库或 APP；
2. 直接联系品牌确认是否使用动物成分；
3. 选择带有纯素认证的产品，以降低误食风险。

### 4. 常见食品添加剂（需关注纯素与健康）

| 添加剂 | 类型 | 可能来源 | 留意事项 |
|:----:|:----|:----|:----|
| **甜蜜素** | 人工甜味剂 | 化学合成 | 多存在于无糖饮料、糖果，建议控制摄入量 |
| **安赛蜜** | 人工甜味剂 | 化学合成 | 与阿斯巴甜共用时甜味更高，儿童与孕妇建议少量 |
| **阿斯巴甜** | 人工甜味剂 | 化学合成 | 苯丙酮尿症患者需避免（含苯丙氨酸） |
| **山梨酸钾/苯甲酸钠** | 防腐剂 | 合成或天然 | 常用于调味酱、饮料，过量可能刺激肠胃 |
| **亚硝酸盐** | 抗氧化、防腐 | 合成 | 腌制品常见，建议严格控制摄入量 |
| **明胶** | 增稠剂 | 动物骨骼、皮肤 | 属于动物成分，纯素者需避免，可改用琼脂、果胶 |
| **乳化剂** | 乳化剂 | 植物油或动物脂肪 | 部分品牌取自动物脂肪，需确认来源 |
| **天然香料/香精** | 香料 | 植物或动物 | “天然”并不代表纯素，疑似时可咨询厂商 |

> 小贴士：看到陌生 E 编号或化学名时，可使用食品添加剂查询网站或 APP；无法确认是否植物来源时，主动询问品牌客服，避免误食动物成分。

## 如何判断是否纯素

### 1. 查看认证标志

**纯素认证：**
- **Vegan Society标志**（英国）
- **V-Label**（欧洲）
- **PETA认证**

**注意：**
- 没有认证不代表不是纯素
- 需要结合配料表判断

### 2. 仔细阅读配料表

**方法：**
- 从头到尾仔细阅读
- 注意不熟悉的成分
- 如果不确定，查找或询问

### 3. 注意"可能含有"（May Contain）

**含义：**
- 通常指生产过程中可能接触到
- 对于过敏人群重要
- 对于纯素者，通常可以接受（如果只是"可能"）

## 营养成分表

### 1. 基础信息

**通常包含：**
- 每份含量
- 每100克/毫升含量
- 能量（卡路里）
- 蛋白质、脂肪、碳水化合物
- 纤维
- 钠
- 某些维生素和矿物质

### 2. 如何阅读

**每份 vs 每100克：**
- **每份：** 根据包装建议的份量
- **每100克：** 标准化，便于比较

**建议：**
- 比较不同产品时，使用"每100克"
- 了解实际摄入时，使用"每份"

### 3. 关键指标

| 指标 | 建议值 | 注意事项 |
|:----:|:------|:--------|
| **钠** | 每100克不超过0.6克盐（2.4克钠） | 某些加工食品钠含量很高 |
| **糖** | 每100克不超过5克糖 | 包括天然糖和添加糖 |
| **总脂肪** | 关注总量 | - |
| **饱和脂肪** | 每100克不超过5克 | 越少越好 |
| **反式脂肪** | 应该为0 | - |
| **纤维** | 每100克至少3克 | 高纤维食物更健康 |

## 特殊标识

### 1. 过敏原标识

**常见标识：**
- 可能含有坚果、大豆、麸质等
- 对过敏人群重要
- 纯素者通常可以忽略（如果只是过敏原）

### 2. 有机标识

**含义：**
- 使用有机方法生产
- 不使用化学农药和化肥
- 可能有认证标志

**注意：**
- 有机不等于纯素
- 仍然需要检查配料表

### 3. 无麸质标识

**含义：**
- 不含麸质
- 适合麸质过敏人群

**注意：**
- 无麸质不等于纯素
- 仍然需要检查配料表

## 实用技巧

#### 1. 快速检查清单

**纯素检查：**
- [ ] 没有明显的动物成分
- [ ] 没有明胶、凝乳酶
- [ ] 没有蜂蜜（如果是纯素）
- [ ] 没有可疑的添加剂

#### 2. 学习常见成分

**方法：**
- 制作清单
- 记住常见的动物成分
- 遇到不熟悉的成分，查找

#### 3. 使用手机应用

**推荐应用：**
- **Is It Vegan?**：扫描条形码
- **Bunny Free**：检查是否零残忍
- **HappyCow**：查找素食餐厅

### 4. 询问商家

**方法：**
- 如果不确定，询问商家
- 查看官方网站
- 联系客服

## 常见陷阱

#### 1. "天然"不等于纯素

**陷阱：**
- 某些"天然"成分可能来自动物
- 例如：明胶、胭脂虫红

#### 2. "植物性"不等于纯素

**陷阱：**
- 可能含有蛋、奶、蜂蜜
- 需要仔细检查配料表

#### 3. 品牌变化

**陷阱：**
- 配方可能改变
- 需要定期检查
- 即使以前是纯素的，现在可能不是

## 小结

学会阅读食品标签和配料表，是素食者必备的技能。通过理解配料表的排序规则，识别隐藏的动物成分，你就能做出明智的购买决定。

> 🏷️ "Read the label, know what you're eating, and make informed choices."
>
> 阅读标签，知道你在吃什么，做出明智的选择。

记住：**仔细阅读，持续学习**，让标签成为你的朋友，而不是敌人！
